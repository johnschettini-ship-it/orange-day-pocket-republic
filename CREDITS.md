# Credits — Orange Day: Pocket Republic

## Game
**Orange Day: Pocket Republic** — cozy satirical civic life-sim.

Design, systems, writing, pixel art pipeline, and code: project author + AI-assisted development (Grok / xAI build sessions).

## Satire notice
All characters are **fictional parody archetypes**. No real names, likenesses, slogans, or campaign logos. Humor targets bureaucracy, media cycles, wealth theater, and campaign spectacle. Mayor Mandate and Leon Rocket are local/tech roles — not presidential candidates.

## Technology
- HTML5 Canvas, vanilla JavaScript & CSS  
- Content module `data.js` + runtime `game.js`  
- Procedural pixel sprites (`tools/gen_sprites.py` + Pillow)  
- Web Audio API procedural soundtrack (no third-party samples)  
  Themes: cold-open · title · character select · play (per-district pads) · evening · election night  
  Stings: composure burst · achievement chime · election fanfare  
 

## Fonts
System UI stack: Segoe UI / system-ui (no bundled font license required).

## Tools
- Browser runtime (Chrome, Edge, Firefox recommended)  
- Optional local server: `python -m http.server`  
- Headless QA: `node smoke.js`
