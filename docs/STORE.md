## Title
Orange Day: Pocket Republic

## Short description (itch blurb)
Tiny citizens. Oversized town. Big civic drama.

A cozy satirical life-sim where you hustle Pocket Republic for a full **election week**: deliver permits, recruit voter blocs, form coalitions, survive scandals, and shape Election Night.

**Build:** v1.3 Full Week Ship — multi-slot save, glossary, achievement gallery, polished title art.

## Long description

**Orange Day: Pocket Republic** mixes errand life-sim energy with systems satire—think cozy plaza loops meeting campaign theater, without real politicians on the cast list.

### You will
- Play as six cartoon **civic archetypes** (organizer, mayor, inventor, tycoon, and more)
- Explore **four districts** unlocked across the week (Plaza, Media Alley, Campus Green, Donor Heights)
- Collect **12 voter groups**, manage loyalty and **rival spats**
- Build **coalitions** that change prices, heat, and endings
- Face daily **crises**, board rules, and setpieces (Debate, Leak, March, Gala)
- Close the week on **Election Night** with multiple outcomes

### Controls
| Input | Action |
|--------|--------|
| WASD / arrows / gamepad stick | Move |
| E / gamepad A | Interact / talk |
| Q / gamepad X | Power (or buy upgrade at vending) |
| Tab | Objectives |
| C | Voter codex |
| G | Achievement gallery |
| H | Glossary |
| O | Options (audio, text size, reduce flash, day length) |
| 1 / 2 / 3 | Save slots (title) |
| Esc / Start | Pause |
| HOME | Sleep when allowed → next day |

### Satire disclaimer (required)
All characters are fictional parody archetypes. This game does **not** use real names, likenesses, slogans, or campaign logos. Mayor and inventor roles are local/tech satire—not presidential candidates.

### Tech
Browser game. Unzip and open `index.html`, or host the folder statically. No install.
