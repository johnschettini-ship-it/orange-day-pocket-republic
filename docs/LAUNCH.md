# Launch checklist — Orange Day: Pocket Republic v1.3

**Build:** `v1.3` · **Zip:** `releases/OrangeDay-PocketRepublic-v1.3.zip`

## Play locally
1. Open `index.html` in Chrome or Edge (or `python -m http.server` in this folder).  
2. Hard-refresh (**Ctrl+F5**) so **v1.3** loads.  
3. Confirm title shows **Full Week Ship · v1.3**.  
4. Optional: `node smoke.js` — expect **GO**.

## Upload to itch.io
1. Create project → **HTML** or **Downloadable** (zip of static files).  
2. Upload: `releases/OrangeDay-PocketRepublic-v1.3.zip`  
3. Paste blurb from `docs/STORE.md`.  
4. Include **satire disclaimer** from STORE.md on the page.  
5. Tags from STORE.md.  
6. Cover: title key art / title screen capture (`assets/ui/key_art.png`).

## What’s in the zip
- Game: `index.html`, `style.css`, `data.js`, `game.js`, `assets/`  
- Docs: `README.md`, `CREDITS.md`, key `docs/*` (STORE, ENDINGS, PLAYTEST, LAUNCH)
